# Funcionalidades Bonus (+2 puntos)

Implementar una o más de las siguientes características para obtener puntos bonus:

## Opciones Bonus

- Simulación de eventos aleatorios durante las misiones (tormentas solares, fallos mecánicos, descubrimientos inesperados)
- Sistema de niveles para los astronautas según su experiencia
- Barra de progreso animada durante la ejecución de la misión
- Modo oscuro/claro
- Panel de filtros por especialidad, estado o experiencia
- Ranking de los astronautas más experimentados
- Indicadores visuales con colores según el estado de energía y salud
- Animaciones suaves en tarjetas y paneles utilizando Tailwind CSS

## Nota

El puntaje máximo continúa siendo 20 puntos. Las funcionalidades bonus podrán compensar pequeñas deficiencias o servir como criterio de desempate entre proyectos.
