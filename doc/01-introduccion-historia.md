# Introducción e Historia - Operación Nova Horizon

## Introducción

La humanidad ha logrado establecer una base científica en el planeta Nova Horizon, ubicado a cientos de años luz de la Tierra. Sin embargo, la colonia depende de expediciones periódicas para obtener recursos, investigar nuevas zonas y garantizar la supervivencia de sus habitantes.

El objetivo de este proyecto es desarrollar un sistema que permita administrar una expedición espacial completa. El usuario será el comandante de la misión y deberá tomar decisiones estratégicas para conformar el equipo de astronautas, administrar los recursos disponibles y seleccionar las misiones más adecuadas.

## Contexto

La colonia Nova Horizon cuenta con recursos limitados. Cada misión implica riesgos y consume recursos. Los astronautas poseen habilidades diferentes. Elegir un equipo inadecuado puede provocar el fracaso de la expedición. El comandante deberá administrar correctamente todos los recursos disponibles para mantener la colonia funcionando.

## Historia

La Agencia Espacial Internacional ha detectado señales provenientes de una zona inexplorada. Se sospecha que allí existen enormes reservas de minerales capaces de solucionar la crisis energética de la Tierra. Sin embargo, la región presenta condiciones extremadamente peligrosas.

La agencia necesita un comandante capaz de organizar expediciones exitosas. Cada decisión afectará el desarrollo de la misión.

## Tecnologías a Utilizar

Todo el sistema deberá desarrollarse utilizando únicamente:

- **HTML5**
- **Tailwind CSS 4.3**
- **CSS** (si es necesario)
- **JavaScript Vanilla**

**No se permite utilizar bases de datos ni tecnologías externas.**

## Alcance del Sistema

El sistema permitirá administrar:

- Astronautas
- Recursos
- Misiones
- Resultados
- Estadísticas

Todo deberá realizarse desde una única aplicación web.
